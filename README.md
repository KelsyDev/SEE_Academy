# 🎉 Fun Calculator 🎉

Welcome to the **Fun Calculator**! This simple Python program lets you add, subtract, multiply, and divide two numbers with a sprinkle of fun and some cool emojis! 😎✨

---

## Features
- Accepts two numbers (including decimals)
- Computes:
  - Sum ➕
  - Difference ➖
  - Product ✖️
  - Quotient ➗
- Displays all results in a clear, friendly format

---

## How to Use

1. Run the Python script.
2. When prompted, enter the first number (decimals allowed).
3. Enter the second number.
4. View the results for sum, difference, product, and quotient.

---

## Important Notes
- The program uses `float()` to accept decimal inputs.
- **Division by zero is not handled** in this version—please do not enter 0 as the second number when dividing to avoid errors.
- This is a simple educational script and meant for quick calculations.

---

## Example

```
Enter the first number: 10.5
Enter the second number: 2

Results of your two numbers:
Sum: 12.5
Difference: 8.5
Product: 21.0
Quotient: 5.25
```

---

## Code Overview

The program is structured as follows:
- Takes user input for two numbers
- Performs arithmetic operations: addition, subtraction, multiplication, division
- Prints the results with fun, descriptive messages and emojis
